  • Application for bird observersmore_vert

    PHP, Symfony, Twig, Bootstrap4, Javascript, Gulp

    Take a look!
    Application for bird observersclose

    This application, entirely built with Symfony aims at providing a mobile first application to record birds observation at the user location.
    A user can record on-site observation of a specific species (a list is already uploaded in the database via a csv file from the offical TAXREF document). The observation is then submitted to review by curators called Naturalists before being officially published on the website. Therefore the application has a complete backend administration dashboard customized according to the user's rights
    The observations can be retrieved easily by location, species or name thanks to a filtered search function.

    This project was part of my online courses in web development at OpenClassrooms.

    The technologies used are Symfony, a PHP framework, the template engine Twig, Bootstrap4, SASS, Javascript for the front-end. The taskrunner Gulp was used to optimized the assets.
    The location of the observations is managed with HTML5 and GoogleMap API.
    The typehint functionnality is implemented with TypeAhead and BloodHound.

  • Online ticketing service for the Louvre Museummore_vert

    PHP, Symfony, Twig, Bootstrap and Jquery

    Take a look!
    Online ticketing service for the Louvre Museumclose

    This application, entirely built with Symfony is a robust system engineered to provide a ticketing service for the Louvre Museum.
    It allows a user to book one or several tickets for the present day (or a future date) if the museum is open on that day. It also automatically renders the price according to the age of the ticket holder. Once the booking is finished a templated email including the booking information is sent to the buyer.

    This project was part of my online courses in web development at OpenClassrooms.

    The technologies used are Symfony, a PHP framework, the template engine Twig, Bootstrap3, Jquery and JQueryUI for the front-end. The payment process is managed by the Stripe API.
