    Online ticketing service for the Louvre Museumclose

    This application, entirely built the PHP framework Symfony is a robust system engineered to provide a ticketing system for the Louvre Museum.
    It allows a user to book one or several tickets for the present day (or a future date) if the museum is open day. It also automatically renders the price according to the age of the ticket holder. Once the booking is finished a templated email including the booking information is sent to the buyer.

    This project was part of my online courses in web development at OpenClassrooms.

    The technologies used are Symfony, a PHP framework, the template engine Twig, Bootstrap3, Jquery and JQueryUI for the front-end. The payment process is managed by the Stripe API.
